# API REST Using Artificial Inteligence
This API REST was developed during the NLW event (Rocketseat) and the purpose is to generate transcriptions for videos.

## Technologies used:
- Node.js
- Fastify
- Prisma
- Zod
- SQLite
- Openai API

## Requests:
- Create Transcription
- Generate AI Completion
- Get All Prompts
- Upload Video
